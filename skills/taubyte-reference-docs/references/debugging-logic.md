# Debugging Logic

## Debug Mode

- Function call logging enabled
- Logs written to temp directory
- Captures function execution details
- Useful for debugging function behavior

## Status Checking

- Universe status: Shows all nodes, protocols, ports
- Service status: Shows service details and URLs
- Universe ID: Unique identifier for universe

## Common Errors

- Name required: Provide name as argument or flag
- Argument order: Flags before arguments or arguments first
- Enable/disable conflict: Use only one flag
- Duplicate ports: Each port must be unique
- Universe state: Universe must be stopped for modifications
- Service not found: Verify service name and existence
- Disabled service binding: Enable service before binding
