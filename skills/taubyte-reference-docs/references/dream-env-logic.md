# Dream Environment Logic

## Dream Multiverse

- Local Taubyte network environment
- Runs on localhost
- Uses project's current branch
- Can be cached for reuse

## Building Resources

- Config: Builds project configuration
- Code: Builds application code
- Function: Builds function code
- Website: Builds website content
- Library: Builds library code

## Attaching vs Injecting Projects

- Attach: Links project to Dream without modifying universe state
- Inject: Injects project data into universe (more invasive)
