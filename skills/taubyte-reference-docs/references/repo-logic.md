# Repository Logic

## Repository Types

- Project repositories: Config and code repositories
- Library repositories: Shared code repositories
- Website repositories: Website content repositories

## Repository Operations

All repository types support:
- Clone: Download repository locally
- Push: Upload local changes
- Pull: Download remote changes
- Checkout: Switch branches
- Import: Import existing repository

## Repository Naming

- Projects: `tb_config_<name>` and `tb_code_<name>`
- Libraries: `tb_library_<name>`
- Websites: `tb_website_<name>`
