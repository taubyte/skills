# Fixture Logic

## Fixtures Overview

Fixtures are pre-configured scenarios that set up test data, projects, repositories, and other resources in a universe.

## setBranch

Sets the default branch for services to resolve.

## pushAll

Pushes all repositories to the universe.

## attachPlugin

Injects plugin binaries built using VM-Orbit into the substrate service.

## pushSpecific

Pushes specific repositories to the universe.
